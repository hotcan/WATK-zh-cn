2012年11月更新

Windows Azure培训包2012年11月份更新包括52个实践，24个演示和37个课程，用以帮助你了解如何使用Windows Azure服务创建应用程序。此外，11月份还更新了实践中使用Windows Azure SDK最新版 (2012年10月版)的内容。

内容更新:

  • 更新的实践以使用Windows Azure SDK (2012年10月版)
  • 更新的实践以使用Visual Studio 2012和Windows 8
  • 新的实践:：Windows 8移动服务 (HTMLJS)
  • 更新的课程：NET 4.5在网站和云服务上的支持
