Windows Azure 培训包

Windows Azure培训包包含了实践、课程和范例，以帮助你理解如何使用Windows Azure服务创建应用程序。
请注意: 本培训包中的实践需要一个Windows Azure账户。如果你没有账户的话，你现在就可以注册免费试用账户。

Windows Azure是什么?

Windows Azure是一个开放灵活的云平台，让你可以快速地创建、部署和管理遍布在托管给微软的数据中心上部署的全球网络上的应用程序。你可以使用任何语言，工具或框架来创建应用程序。并且你可以将你的公有云应用程序与你现有的IT环境相结合。
