Prerequisites

The hands-on labs included in the training kit are designed to use either Visual Studio 2010, Visual Studio 2012, or Mac OS X. These hands-on labs have the following common pre-requisites.

    Visual Studio 2010

  • Visual Studio 2010
  • Microsoft .NET Framework 4.0
  • Windows Azure SDK

    Visual Studio 2012

  • Visual Studio 2012
  • Microsoft .NET Framework 4.5
  • Windows Azure SDK

    Mac OS X

  • Git
  • Node.js
  • Windows Azure Command-Line Tools for Mac
